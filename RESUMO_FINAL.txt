🚀 COMO COMPILAR PARA ANDROID
═══════════════════════════════════════════════════════════════════════════════

Opção 1: Build Local (Rápido, apenas debug)
──────────────────────────────────────────
  cd /home/user/streamflix-v2.0.4
  bun install
  bun run android

  ⚠️  Requer Android Studio + emulador ou device USB
  ⏱️  Tempo: ~5-10 minutos


Opção 2: Build EAS (Produção, recomendado)
────────────────────────────────────────────
  cd /home/user/streamflix-v2.0.4
  bun install
  
  # Primeiro, faça login
  eas login
  
  # Depois compile
  eas build --platform android
  
  # Opcionalmente, envie direto pra Play Store
  eas build --platform android --auto-submit

  ⏱️  Tempo: ~10-15 minutos (na nuvem)
  📱 Gera: APK + AAB (pronto pra Play Store)


Opção 3: Rodar em Desenvolvimento (sem build)
──────────────────────────────────────────────
  cd /home/user/streamflix-v2.0.4
  bun run start
  
  # Escaneie QR com Expo Go (Play Store)
  # Ou: bun run android para emulador

  ⏱️  Tempo: ~2-3 minutos

🎨 CUSTOMIZAÇÕES FÁCEIS
═══════════════════════════════════════════════════════════════════════════════

Mudar cor vermelha:
  Arquivo: constants/colors.ts
  Mude: primary: '#E50914'  ← para sua cor hex


Mudar API key TMDB:
  Arquivo: services/api.ts
  Mude: const TMDB_API_KEY = 'sua_chave_aqui'
  Obtenha em: https://www.themoviedb.org/settings/api


Mudar idioma de português:
  Arquivo: services/api.ts
  Mude: language: 'pt-BR'  ← para outro código de idioma
  Exemplos: 'en-US', 'es-ES', 'fr-FR'


Adicionar nova aba:
  1. Edite: app/(tabs)/_layout.tsx
  2. Adicione: <Tabs.Screen name="minha-aba" ... />
  3. Crie arquivo: app/(tabs)/minha-aba.tsx

🌐 APIs UTILIZADAS
═══════════════════════════════════════════════════════════════════════════════

1. TMDB (The Movie Database)
   • URL: https://api.themoviedb.org/3
   • Dados: Filmes, séries, generos, cast, ratings
   • Key já configurada (limite: 40 requests/10s)
   • Filtros: Região Brasil, idioma PT-BR

2. VidSrc.to
   • URL: https://vidsrc.to/embed/...
   • Reprodução: Filmes e séries sem anúncios
   • Sem API key (embed direto no WebView)
   • Suporte: Full-screen, controles de player

3. AsyncStorage
   • Local: Armazena favoritos no celular
   • Persistência: Dados salvos mesmo desligando
   • Sincronização: Automática com Zustand

🆘 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

❌ "Cannot read property 'colors' of undefined"
✅ RESOLVIDO! (Este era o erro original - corrigido em v2.0.4)

❌ App fecha ao abrir
✅ Solução:
   • bun install (reinstale dependências)
   • bun run android --reset-cache (limpe cache)
   • Atualize Node.js para 20+

❌ Vídeo não carrega
✅ Causa: VidSrc.to pode estar lento
   • Tente novamente em alguns minutos
   • Verifique internet
   • Alguns vídeos podem não estar disponíveis

❌ Busca muito lenta
✅ Ajuste debounce em: app/(tabs)/busca.tsx
   setTimeout(() => performSearch(), 500);  ← mude 500 para menos

❌ Favoritos desaparecem
✅ Limpe AsyncStorage:
   • Desinstale e reinstale app
   • Ou execute: await AsyncStorage.clear()
